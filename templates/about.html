About this project

Website’s origin story?

Since the beginning of time, people have always tried to find the fastest way from point A to point B.

Ivan Mezhibovskiy (H1) and I were no different, debating almost every other day, on which route was faster to the science building - the one next to house 8 or the one by the memorial library?

This sparked a lightbulb moment! I decided to embark on a mission of utmost importance, a project that would shake the very foundations of our daily commute - to conclusively determine which path, in all its glory, is the fastest route to any destination within the campus.

The Nitty-Gritty of interactive map shenanigans

Python Flask is used as the web framework for this project, with the map being rendered using Leaflet.js.

Node data is stored in a static JSON file, while staff and report system data is stored on a secure MongoDB database.

The shortest path algorithm is implemented using Dijkstra's algorithm, on the node data.
